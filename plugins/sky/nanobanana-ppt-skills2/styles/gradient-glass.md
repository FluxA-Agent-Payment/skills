## Page Type Templates

### Cover Page Template
Composition logic: Place a large complex 3D glass object in the center, overlaid with bold text, with extended aurora waves in the background.

Use case: First page of PPT, displaying title and theme.

### Content Page Template
Composition logic: Use Bento grid layout, place 3D icons in small cards, text in large cards. Container materials must be frosted glass with blur effects, featuring refined white edges and soft shadows, with mandatory large internal whitespace to avoid crowding.

Use case: Displaying core insights, key points, content chapters, etc.

### Data Page Template
Composition logic: Use split-screen design, text layout on the left, large floating glowing 3D data visualization charts on the right. Charts should use glowing 3D donut charts, capsule-shaped progress bars, or floating numbers, looking like glowing neon toys.

Use case: Displaying data, statistics, comparative analysis, summaries, etc.
